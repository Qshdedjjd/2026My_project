#  ANTIGRAVITY – 未來感懸浮 UI 實作
### CSS Floating Animation & Futuristic UI Concept

這是一個專注於 **CSS 微互動 (Micro-interactions)** 與 **空間美學** 的前端實驗專案。透過純 CSS 實作動態懸浮效果，模擬無重力狀態下的介面交互感受。

##  技術核心與亮點 (Technical Highlights)

### 1. 無重力懸浮動畫 (Gravity-Free Animation)
* **技術實作**：利用 `@keyframes` 結合 `translateY` 與 `rotate`，實作非線性的懸浮路徑。
* **層次感設計**：透過不同的 `animation-duration` (float, float-slow) 與 `animation-delay` (float-delay)，營造出卡片之間不規則、自然流動的視覺層次，避免死板的同步移動。

### 2. 空間美學 UI (Futuristic Aesthetics)
* **視覺風格**：採用 **Deep Space (深空)** 風格背景，搭配玻璃擬態 (Glassmorphism) 或簡約線條，強化「ANTIGRAVITY」的品牌概念。
* **響應式佈局**：使用 Flexbox/Grid 確保卡片在不同螢幕尺寸下皆能保持和諧的星團分佈。

### 3. 使用者體驗細節 (UX Details)
* **Hover 互動**：實作滑鼠懸停時的緩衝停止或放大效果，增強介面的可操作性反饋。
